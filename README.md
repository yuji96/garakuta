## algorithms

### suudoku（数独）

数独を解く脳筋アルゴリズム。再帰の勉強用に使ったのだろう。単純に枝刈りしながら再帰的に総当りしている。
ただの Python だと実行が終わらないが、numba を使用することで最小ヒント問題でも 3 分以内に解けるようになった。
数独のテクニックを実装すれば何倍も早くなるらしい。

```
python main.py
```
