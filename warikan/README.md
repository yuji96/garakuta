# 割り勘精算機

割り勘精算の最小手を計算する。LINE bot にしたら需要ありそうだけど今やる気はない。Google フォームと GAS にした方が UX よさそう。

## input

- 支払い者
- 支払額
- 金額指定

```csv
payer,amount,note
A,10000,
B,7000,
C,3500,
D,12000,
B,3000,"{""A"": 2000, ""B"": 1200, ""C"": 1000, ""D"": 2000}"
```

## output

```
   A     B  C     D
A  0     0  0   125
B  0     0  0     0
C  0  3875  0  1750
D  0     0  0     0
```
